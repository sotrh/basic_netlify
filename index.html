Display lists

You can create bullet point lists with <ul>. You use <li> for each item in the list.

For example:


        <ul>
            <li>This is an item</li>
            <li>This is also an item</li>
            <li>So is this</li>
        </ul>

This will render the following

  • This is an item
  • This is also an item
  • So is this

We can create numbered lists with <ol>. We still use <li> for each item in the list.


        <ol>
            <li>This is the 1st item</li>
            <li>This is the 2nd</li>
            <li>And this is the 3rd</li>
        </ol>
  1. This is the 1st item
  2. This is the 2nd
  3. And this is the 3rd
